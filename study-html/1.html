HTML(Hypertext Markup Language) 은 웹 페이지 및 웹 응용 프로그램을 만들기 위한 표준 마크업 언어입니다.

웹 브라우저는 웹 서버 또는 로컬 저장소에서 HTML 문서를 수신하여 멀티미디어 웹 페이지로 렌더링합니다.

HTML은 의미론적으로 웹 페이지의 구조를 설명하고 원래 문서의 모양에 대한 단서를 포함했습니다.

HTML 요소는 HTML 페이지의 빌딩 블록입니다.

HTML 구성을 사용하면 대화형 양식과 같은 이미지 및 기타 개체가 렌더링된 페이지에 포함될 수 있습니다.

제목, 단락, 목록, 링크, 인용문 및 기타 항목과 같은 텍스트에 대한 구조적 의미를 표시하여 구조화된 문서를 생성하는 수단을 제공합니다.

HTML 요소는 꺾쇠 괄호를 사용하여 작성된 태그로 구분됩니다.
