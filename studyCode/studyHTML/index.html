HyperText Markup Language(HTML)는 웹 페이지를 만드는 데 사용되는 표준 마크업 언어이다.

HTML은 CSS, 자바스크립트와 함께 시각적으로 매력적인 웹 페이지, 웹 애플리케이션을 위한 사용자 인터페이스, 수많은 모바일 애플리케이션을 위한 사용자 인터페이스를 만들기 위해 대부분의 웹 사이트에 사용되는 초석 기술이다. 웹 브라우저는 HTML 파일을 읽고 볼 수 있는 또는 들을 수 있는 웹 페이지로 렌더링할 수 있습니다. HTML은 웹 사이트의 구조를 의미론적으로 표현하기 위한 신호와 함께 기술하여 프로그래밍 언어가 아닌 마크업 언어로 만든다.

HTML 요소는 모든 웹 사이트의 구성 요소를 구성합니다. HTML은 이미지와 객체를 삽입할 수 있게 하며 대화형 양식을 만드는 데 사용할 수 있습니다. 제목, 단락, 목록, 링크, 인용문 및 기타 항목과 같은 텍스트에 대한 구조적 의미론을 표시함으로써 구조화된 문서를 작성할 수 있는 수단을 제공한다.

언어는 꺾쇠 괄호로 묶인 태그로 구성된 HTML 요소의 형태로 작성됩니다. 브라우저는 HTML 태그와 스크립트를 표시하지 않고 페이지의 내용을 해석하는 데 사용합니다.
