css 색상값 예시

1. color name

미리 정의된 색상 이름을 사용하여 색상을 설정

red green blue

2. RGB

rgb(빨강, 초록, 파랑)의 형식을 사용하여 색상을 설정

0~255 사이의 색상 강도를 가진다.

rgb(238, 130, 238) rgb(255, 165, 0) rgb(60, 179, 113)

3. HEX

#rrggbb의 형식을 사용하여 색상을 설정

00~ff 사이의 16진수 값으로 색상 강도를 가진다.

#af7812 #f35bf3 #4536a5

4. HSL

hsl(색조, 채도, 밝기)의 형식을 사용하여 색상을 설정

색조는 0~360도의 색상환, 채도는 0~100%의 회색 음영, 밝기는 0~100%의 빛의 양을 의미한다.

hsl(248, 53%, 58%) hsl(147, 50%, 47%) hsl(0, 100%, 50%)
