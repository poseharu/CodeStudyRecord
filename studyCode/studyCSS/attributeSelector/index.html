Attribute Selector

1. [attribute] Selector

class 속성을 가진 요소를 선택하여 노란표시

yellow yellow color yellow-color yellowcolor color yellow color-yellow coloryellow

2. [attribute="value"] Selector

class 속성값이 'yellow'인 요소를 선택하여 노란표시

yellow yellow color yellow-color yellowcolor color yellow color-yellow coloryellow

3. [attribute~="value"] Selector

class 속성값에서 'yellow'이란 단어가 포함된 요소를 선택하여 노란표시

(전체 단어여야함)

yellow yellow color yellow-color yellowcolor color yellow color-yellow coloryellow

4. [attribute|="value"] Selector

class 속성값에서 'yellow'으로 시작하는 값을 가진 요소를 선택하여 노란표시

(단독으로 사용하거나 하이픈(-)이 뒤에 오는 전체 단어여야함)

yellow yellow color yellow-color yellowcolor color yellow color-yellow coloryellow

5. [attribute^="value"] Selector

class 속성값에서 'yellow'으로 시작하는 값을 가진 요소를 선택하여 노란표시

(전체 단어일 필요는 없음)

yellow yellow color yellow-color yellowcolor color yellow color-yellow coloryellow

6. [attribute$="value"] Selector

class 속성값에서 'yellow'으로 끝나는 값을 가진 요소를 선택하여 노란표시

(전체 단어일 필요는 없음)

yellow yellow color yellow-color yellowcolor color yellow color-yellow coloryellow

7. [attribute*="value"] Selector

class 속성값에서 'yellow'이 들어가는 값을 가진 요소를 선택하여 노란표시

(전체 단어일 필요는 없음)

yellow yellow color yellow-color yellowcolor color yellow color-yellow coloryellow
