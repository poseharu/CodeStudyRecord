css 단위 예시

1. %단위

상위 요소의 크기를 기준으로 하여 백분률로 나타냄(기본값 16px)

div요소에 font-size: 20px 설정함.

div요소 밖: 150% = 16 * 1.5 = 24px

div요소 안: 150% = 20 * 1.5 = 30px

2. em 단위

상위 요소의 글꼴 크기에 비례(기본값 16px)

div요소에 font-size: 20px 설정함.

div요소 밖: 1.5em = 16 * 1.5 = 24px

div요소 안: 1.5em = 20 * 1.5 = 30px

3. rem 단위

루트 요소(html요소)의 글꼴 크기에 비례(기본값 16px)

div요소에 font-size: 20px 설정함.

div요소 밖: 1.5rem = 16 * 1.5 = 24px

div요소 안: 1.5rem = 16 * 1.5 = 24px

4. vw 단위

1vw = 실제 화면 너비의 1/100

3vw

5. vh 단위

1vh = 실제 화면 높이의 1/100

3vh

6. vmin

vw 와 vh 중에서 작은 크기를 선택하여 적용

3vmin

7. vmax

vw 와 vh 중에서 큰 크기를 선택하여 적용

3vmax
